Open time-management-review.html with Chrome, Edge, Safari, or Firefox.

If the browser shows a blank page because local file scripts are blocked, use the local-server version:
1. Double-click start-review.command on macOS.
2. The browser should open http://127.0.0.1:8080/dist/ automatically.

Manual fallback:
1. Open a terminal in this folder.
2. Run: python3 -m http.server 8080
3. Visit: http://127.0.0.1:8080/dist/
